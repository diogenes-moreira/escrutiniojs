Escrutiniojs

Es la version en Sails.js de la aplicación de escrutinio, la persistencia es relacional, usa mysql

Download this project as a .zip file Download this project as a tar.gz file

Esto es una versión del sistema de escrutinio que usamos en Quilmes varias veces pero escrita con node.js No tiene seguridad porque hasta el momento no hizo falta.

Deja los datos en una base relación, mysql, para después explotarlos con un cubito olap o alguna planilla de calculo.

voy a dejar los pasos de instalación para alguna distro debian, usan la que quieran. el tema es llegar al npm, a partir de ahí es todo igual. Estoy partiendo de la base que ya se tiene instalado el mysql, sino es tan facil como hacer

sudo apt-get install mysql-server

arranquemos

sudo apt-get install python-software-properties python g++ make sudo add-apt-repository ppa:chris-lea/node.js sudo apt-get update sudo apt-get install nodejs

Luego hay que tirar los siguiente comandos

sudo npm install -g sails sudo npm install -g mysql sudo npm install -g grunt sudo npm install -g sails-mysql

luego hacen checkout del proyecto y listo...

en /conf/application.js estan las configuraciones de puerto y ruta donde escucha la aplicacion en /conf/adapters.js estan las conexiones de DB

¿como se deploya? le dejo el link a sails.js

hay un controler /initializer y /initializer/etapa2 que genera la configuración para usarser.. esto es valido para quilmes pero sirve como ejemplo.. cada distrito puede generarse el suyo y subirlo :)

la carga de la planilla de escrutinio /carga

!ojo! si el usuario con el que corre sails no tiene permisos de escritura sobre el directorio, generenle un /.temp con permisos para r/w para ese usuario. es lo que usa nodejs y sailsjs para sus archivos temporales.

ya tiene todo para usar socket.io.js, pero no lo use porque no hizo falta.. para la plataforma esta prepara para usarse.
